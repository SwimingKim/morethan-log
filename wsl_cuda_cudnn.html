1. 시스템 명령어 활성화하기

  • WSL에서 su 혹은 sudo 명령어를 사용하기 위해서는 wsl.conf 설정이 필요하다.
    • 사전에 설정이 되어있다면 다음 단계로 넘어간다.
  • WSL에서 /etc/wsl.conf 파일을 만든 후, 다음의 내용을 추가한다.
      • [boot]
systemd=true
    • 참고 : https://learn.microsoft.com/ko-kr/windows/wsl/wsl-config
  • 파일 생성 후 WSL를 재부팅한다.
    • Powershell 혹은 CMD에서 wsl --shutdown을 실행한다.
 

2. CUDA 설치 (11.6 버전)

  • CUDA 설치 페이지에 접속한다.
  • Linux > x86_64 > WSL-Ubuntu > 2.0 > deb(local) 순서대로 선택한다.
  • 아래 페이지에 적혀있는 바와 같이 명령어를 실행해나간다.
      • wget https://developer.download.nvidia.com/compute/cuda/repos/wsl-ubuntu/x86_64/cuda-wsl-ubuntu.pin
sudo mv cuda-wsl-ubuntu.pin /etc/apt/preferences.d/cuda-repository-pin-600
wget https://developer.download.nvidia.com/compute/cuda/11.6.1/local_installers/cuda-repo-wsl-ubuntu-11-6-local_11.6.1-1_amd64.deb
sudo dpkg -i cuda-repo-wsl-ubuntu-11-6-local_11.6.1-1_amd64.deb
sudo apt-key add /var/cuda-repo-wsl-ubuntu-11-6-local/7fa2af80.pub
sudo apt-get update
sudo apt-get -y install cuda
  • .bashrc에 다음의 내용을 추가한다
      • export PATH=/usr/local/cuda-11.6/bin:${PATH:+:${PATH}}
export LD_LIBRARY_PATH=/usr/local/cuda-11.6/lib64:${LD_LIBRARY_PATH:+:${LD_LIBRARY_PATH}}
  • .bashrc의 변경된 내용을 활성화하고, cuda 버전을 확인한다.
      • source ~/.bashrc
nvcc --version
 

3. CuDNN 설치하기

  • CuDNN 설치 페이지에 접속한 후, NVIDIA 로그인을 한다.
  • Agree 옵션을 활성화한 후, CUDA 버전와 OS에 알맞 installer 파일을 다운받는다.
    • CUDA 11.6와 Ubuntu 20.04(x86_64)에 맞는 설치파일이 필요하면, 빨간색 표시와 같이 다운받으면 된다.
  • 설치 파일을 wsl로 옮긴 후 설치를 진행한다.
    • WSL에서 윈도우의 파일에 접근할 수 있어서, 다음과 같은 명령어로 파일을 불러왔다.
        • # mv /mnt/c/Users/${계정명}/Downloads/cudnn-local-repo-ubuntu2004-8.7.0.84_1.0-1_amd64.deb ./
mv /mnt/c/Users/SBAuser/Downloads/cudnn-local-repo-ubuntu2004-8.7.0.84_1.0-1_amd64.deb ./

sudo dpkg -i ./cudnn-local-repo-ubuntu2004-8.7.0.84_1.0-1_amd64.deb
sudo cp /var/cudnn-local-repo-ubuntu2004-8.7.0.84/cudnn-local-A3837CDF-keyring.gpg /usr/share/keyrings/
sudo apt-get update
sudo apt-get install libcudnn8=8.7.0.84-1+cuda11.8
sudo apt-get install libcudnn8-dev=8.7.0.84-1+cuda11.8
sudo apt-get install libcudnn8-samples=8.7.0.84-1+cuda11.8
    • 참고 : https://docs.nvidia.com/deeplearning/cudnn/install-guide/index.html#installlinux-deb
  • CuDNN 버전을 확인한다.
      • cat /usr/include/x86_64-linux-gnu/cudnn_version_v8.h | grep CUDNN_MAJOR
